## 备考要点

| 要点 | 说明 |
|------|------|
| 填空题优先保正确率 | A-E 全对是拿奖的基础，宁可多花时间验证，不要急着跳编程题 |
| 程序当草稿纸 | 填空题答案可以写暴力程序跑出来，不需要数学推导 |
| 边界条件敏感 | 提交前手推 2-3 组极端样例（最大值、全0、只有1个元素） |
| long long 习惯 | 乘法/累加结果超过 10^9 时必须用 long long，否则溢出 |
| 模板要熟 | 质数筛、快速幂、前缀和、BFS、0-1背包 这5个要达到背默水平 |
